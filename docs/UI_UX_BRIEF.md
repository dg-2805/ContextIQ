# ContextIQ – UI/UX Design Brief

## Design Vision

ContextIQ should feel like a modern AI-native SaaS product inspired by:

* ChatGPT
* Claude
* Perplexity

The interface should prioritize simplicity, clarity, and trust.

Users should be able to:

1. Add knowledge sources quickly
2. Ask questions immediately
3. View supporting evidence
4. Trust the generated responses

---

## Design Principles

### Simplicity

Minimize unnecessary actions.

### Transparency

Every answer should be traceable to its source.

### Consistency

Maintain a consistent layout and interaction model across all pages.

### Accessibility

Support keyboard navigation, readable typography, and responsive layouts.
